Cellipede

... is a python flask app that uses javascript for spreadsheets instead of google's spreadsheets.
It is smaller and faster than a similar app called Pipulate, plus it's coded using best practices.
While most of the functions, scrapers, and APIs are related to SEO tasks, cellipede can be used for other things.
Included is an example that fills a sheet with a daily listing of 'ruby on rails' job postings from oDesk's RSS feed.

Features

  • add your own or use the provided Scrapers and API's for social metrics:
    • Generic scrapers: title, description, ga
    • YouTube: subscribers, total views, video title, views, thumbs up, thumbs down
    • Twitter: tweet total, following, followers, tweets for any url
    • Google +1's
    • Facebook: shares, likes
  • learn to program in python using cellipede as a starter app, which is a great example of best practices in action, but these are highly recommended:
    • sentdex - self-taught python programmer
    • more sentdex
    • Michael Herman's flask tutorial
    • Miguel Grinberg
  • customize it to suit your needs
  • easily collaborate/share with other people via google's spreadsheets in several ways:
    1. in a cellipede sheet do ctrl+a to select all, ctrl+c to copy, then in a google spreadsheet do ctrl+v to paste
    2. to import a whole cellipede sheet into a google spreadsheet:
      =IMPORTHTML("http://???/sheet_as_table?id=?","table",1)
      replace ??? with the url of your cellipede server and the id=? with any cellipede sheet number
    3. to perform a cellipede function in a google spreadsheet cell:
      =IMPORTHTML("http://cellipede.herokuapp.com/sheet_function?afunction=tweets&url=http://www.amazon.com/","table",1)
      replace cellipede.herokuapp.com with the url of your cellipede server, then replace the afunction=? and url=??? with the desired function/url

Please feel free to try the demos and to submit questions/comments/rants as an issue on either of the github repositories.
