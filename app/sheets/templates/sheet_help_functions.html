Functions help

Scrapers   to scrape data from a page at a url

Generic scrape from any page at url

  • ?serps(A1) : serps from google's search result page ... not via the deprecated api
  • ?links(A1) : scrape all links, a href's, from a page
  • ?title(A1) : scrape title from the page
  • ?description(A1) : scrape description from the page
  • ?ga(a1) : scrape google analytics info from the page, which returns the "UA-?-?"

YouTube e.g. https://www.youtube.com/watch?v=paX3Q6ZBuyg so enter paX3Q6ZBuyg into a cell

  • ?title(a1) : scrape the video title for a given video id
  • ?views(a1) : scrape the views count for a given video id
  • ?thumbsup(a1) : scrape the thumbsup count for a given video id
  • ?thumbsdown(a1) : scrape the thumbsdown count for a given video id

Twitter first, enter a twitter user name into a cell

  • ?tweettotal(a1) : scrape the twitter account tweets count for a given user
  • ?following(a1) : scrape the twitter account following count for a given user
  • ?followers(a1) : scrape the twitter account followers count for a given user

API's   to retrieve data, mostly counts, via an API call

Twitter

  • ?tweets(A1) : get count using twitter api for a given url

Facebook

  • ?shares(A1) : get count using facebook api for a given url
  • ?likes(A1) : get count using facebook api for a given url

Google +1's and YouTube

  • ?plusses(A1) : get count using google api for a given url
  • ?subscribers(A1) : get YouTube subscriber count using google api for a given user name
  • ?totalviews(A1) : get YouTube total views count using google api for a given user name
* right-click on a any of the above then select copy and then paste it into a sheet cell
